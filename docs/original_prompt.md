AI-AGENT OPTIMIERUNG (IMMER BEACHTEN)
Das Projekt muss jederzeit optimal für AI-Agents verständlich sein:

- klare Ordnerstruktur

- kurze Architektur-Dokumentation
- deterministische Logik
- keine impliziten Konventionen

- README strukturiert für automatisches Parsing
- Backlog nachvollziehbar pflegen
- Funktionen klar beschreiben
- Konfiguration leicht auffindbar
